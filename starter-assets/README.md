# Starter Assets

These files are optional starting points for the exercise. You may use them,
adapt them, or ignore them and wire the immutable upstream actions directly.

Included here:

- `workflows/onboard-service.yml`: a starter GitHub Actions workflow
- `actions/onboard-service/action.yml`: a local composite action
  that wraps the immutable upstream onboarding flow

Treat these as starter material, not source of truth. Validate the contracts,
read the upstream action documentation, and make deliberate choices about what
to keep, change, or replace.
